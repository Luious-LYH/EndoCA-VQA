# Paper-Facing Results

This directory contains compact, public-safe tables corresponding to the final manuscript.

- `benchmark/endoca_core_results.csv`: 11-model EndoCA-Core comparison.
- `benchmark/endoca_diagnostic_results.csv`: 11-model EndoCA-Diagnostic comparison.
- `asr/asr_results.csv`: four-model EndoCA-Core ASR result summary and macro average.
- `ablation/no_atomic_ablation.csv`: Direct, no-atomic, and ASR macro comparison.

Values are percentages rounded to one decimal place, matching the displayed manuscript tables. Full per-query predictions and unrounded metric JSON files are kept outside the public Git payload because of size and third-party provider terms.
