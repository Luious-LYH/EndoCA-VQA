# EndoCA Data

`manifests/` contains deterministic gzip archives for EndoCA-Core and EndoCA-Diagnostic. Run `python -m endoca.data.prepare` to extract and verify them.

The full upstream Kvasir-VQA-x1 copy and local construction evidence are retained in the authors' archive but ignored by Git. Public users should obtain images from the official Kvasir-VQA-x1 release and follow its CC BY-NC 4.0 license.
